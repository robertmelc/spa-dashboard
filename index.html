Datové Zdroje & Scraping
Architektura sběru živých dat
⚠ Proč ne přímý scraping z prohlížeče?
Booking.com, lázeňské weby a většina portálů blokuje přímé volání z JS (CORS policy). Live data vyžadují backend proxy server. Níže je architektura a ukázkový kód.
Dostupné zdroje dat
ÚZIS ČR — Lázeňská péče ✓ VOLNĚ
PDF publikace + datové soubory ke stažení. Počty pacientů, ošetřovacích dnů, indikace. Úroveň: kraj / obec. URL: uzis.cz/lazenska-pece
ČSÚ — Kapacity HUZ ✓ API/CSV
Otevřená data ČSÚ — počty lůžek, pokojů, zařízení podle obce a kategorie. Filtr „lázeňské ubytování". URL: czso.cz/csu/czso/hromadna-ubytovaci-zarizeni
Booking.com ✗ CORS blokován
Ceny pokojů live. Vyžaduje backend proxy nebo Booking Affiliate API (registrace zdarma). Dává: ceny, hodnocení, dostupnost.
Weby hotelů (scraping) ⚡ BACKEND
Procedury, ceníky, balíčky — scraping přes Puppeteer/Playwright na vašem serveru. Cílové weby: ensana.com, laznejachymov.cz, spa-resort.cz atd.
Google Places API ✓ LIVE (API klíč)
Hodnocení, fotky, otevírací doby. Bezplatně prvních 200 USD/měsíc. Lze volat přímo z JS.
Tourdata.cz ✓ VOLNĚ
Agregovaná data CzechTourism — hosté, přenocování, průměrné ceny. Export do PDF/Excel.
📦 Backend proxy — ukázkový kód (Node.js + Express)
// server.js — backend proxy pro scraping
const express = require('express');
const puppeteer = require('puppeteer');
const cors = require('cors');
const app = express();
app.use(cors({ origin: 'http://localhost:3000' }));

// Endpoint: procedury z webu hotelu
app.get('/api/procedures/:hotelSlug', async (req, res) => {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();

  const urls = {
    'grandhotel-pupp': 'https://www.pupp.cz/cs/procedury',
    'ensana-bristol': 'https://www.ensana.com/bristol/procedury',
    'lazne-jachymov': 'https://www.laznejachymov.cz/lecba'
  };

  await page.goto(urls[req.params.hotelSlug]);
  const data = await page.evaluate(() => {
    // Extrakce z DOM — přizpůsobit dle webu
    return Array.from(document.querySelectorAll('.procedure-item'))
      .map(el => ({
        name: el.querySelector('.name')?.textContent,
        price: el.querySelector('.price')?.textContent,
        duration: el.querySelector('.duration')?.textContent
      }));
  });
  await browser.close();
  res.json(data);
});

// Endpoint: live ceny z Booking
app.get('/api/prices/:hotelId', async (req, res) => {
  // Booking Affiliate API nebo scraping
  const response = await fetch(
    `https://distribution-xml.booking.com/2.6/json/hotels?hotel_ids=${req.params.hotelId}`,
    { headers: { 'Authorization': 'Basic ' + btoa('USER:PASS') } }
  );
  res.json(await response.json());
});

app.listen(4000, () => console.log('Proxy běží na :4000'));
